//本地命令
git init Git仓库初始化
git add . Git变更检索
git commit {-m "Message"} Git版本提交{注释}
git log Git历史版本
git reset --hard [版本Hash} Git版本强制回退 Warning:回退后文件会消失
git branch Git分支查看
git branch [Name] Git新建分支
git checkout [Name] Git切换分支
git merge [Name] Git合并到当前分支
//在线命令
git clone git@github.com:[Username/RepositoryName.git] Git克隆Github在线仓库
git push Git推到线上
git pull Git拉回线下
